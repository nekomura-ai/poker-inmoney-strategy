## Concept

この資料は、ポーカートーナメントでインマネ率を上げるための外部公開用ガイドです。

ただハンドを待つのではなく、テーブル全体を観察し、相手の性質に合わせて戦略を変えます。

> インマネは偶然の到達点ではなく、序盤から作る作戦の結果です。

## Core Idea

### 序盤は、カードを見る時間だけではない

序盤の目的は、大きく勝つことだけではありません。

- よく参加する人は誰か
- コールしすぎる人は誰か
- すぐ降りる人は誰か
- ベットサイズが正直な人は誰か
- 負けた後にプレイが荒くなる人は誰か
- こちらのショーダウンを覚えている人は誰か
- こちらのレンジを考えている人は誰か

**テーブル全体をスキャンして、攻略マップを作る時間です。**

相手が「この人はブラフがある」「この人は強い時しか打たない」「この人は押せば降りる」と思っているなら、その認識そのものが武器になります。

| 10BBから20BB | コールを減らし、先に攻める |
| 10BB未満 | 待ちすぎずプッシュを考える |

## ITM Difference

海外の大型トーナメントでは、ITMが全体の12%から15%前後に設定されることが多くあります。

一方で、日本のアミューズ系・権利型トーナメントでは、実質的な入賞や権利獲得が上位3%から8%前後になることがあります。

| 形式 | ITMの目安 | 基本戦略 |
|---|---:|---|
| 海外大型MTT | 12%から15%前後 | バブルを越える設計が重要 |
| 日本型トーナメント | 3%から8%前後 | バブルに着く前に戦えるスタックを作る |

日本型では、ただ耐えるだけでは届きにくいです。

**インマネ前に、武器を作る。**

---

## Late Registration

「レジクロで入る方がICMが高い」と言われることがあります。

これは半分本当です。すでに飛んだプレイヤーがいるため、入賞までの距離は短くなります。序盤の事故も避けられます。

ただし、弱点もあります。

- 開始時点でスタックが浅くなりやすい
- ポストフロップで実力差を出す時間が少ない
- 1回のオールイン勝負への依存度が高くなる
- 日本型の低ITM率では、ショートで入っても届きにくい

**レジクロ参加はICM効率を上げることがあるが、スタック構築の時間を失う。**

序盤から参加して、観察力・技術力・相手への見せ方を使ってスタックを作れるプレイヤーなら、最初から入る価値は十分あります。

## Target Stack

目標チップは、次の式で考えます。

```text
総チップ量 ÷ ITM人数 = ITMライン付近の平均チップ
```

例:

| 項目 | 数値 |
|---|---:|
| 参加人数 | 100人 |
| スタートチップ | 30,000点 |
| 総チップ量 | 3,000,000点 |
| ITM率 | 5% |
| ITM人数 | 5人 |

```text
3,000,000点 ÷ 5人 = 600,000点
```

上位5人に残るには、終盤で平均60万点前後を持つプレイヤーたちと戦う必要があります。

ただ耐えて10BBで到達するだけでは、かなり苦しくなります。

---

## Final Question

インマネは、ただの結果ではありません。

焦りをコントロールし、相手を観察し、必要な時に一歩前へ出ることで近づく到達点です。

```text
燃料はチップ。
地図はスタックサイズ。
追い風は相手の恐怖と誤認識。
船長は、あなたです。
```

**これは生き残るための一手か。未来のチップを増やす一手か。**

毎ハンド、この問いを持つことが、インマネへの最初の作戦になります。
