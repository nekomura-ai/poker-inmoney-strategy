# ポーカートーナメントでインマネするための作戦資料

## 目的

この資料は、ポーカートーナメントで「入賞圏内まで生き残る確率」を上げるための実戦ガイドです。

ただ長く耐えるのではなく、チップ量、ブラインド、相手の心理、バブルの圧力を読みながら、インマネへの道を設計します。

合言葉はシンプルです。

**序盤は燃料を集め、中盤は事故を避け、終盤は圧力を味方にする。**

---

## インマネに必要な考え方

トーナメントは、キャッシュゲームとは別のゲームです。

キャッシュゲームでは1チップは常に同じ価値に近いですが、トーナメントでは状況によってチップの価値が変わります。特にインマネ直前は「生き残る価値」が大きくなります。

つまり、強いプレイヤーはこう考えます。

- いま増やすべきか
- いま守るべきか
- いま相手にプレッシャーをかけるべきか
- いま無理に勝負してはいけない相手は誰か

インマネは運だけでなく、局面ごとの判断の積み重ねで近づけます。

---

## 1. 序盤：安く情報を集め、強い形でチップを増やす

序盤はブラインドが小さいため、焦って大きな勝負をする必要はありません。

ここでのテーマは、**無理にヒーローにならず、未来の勝負のために地図を作ること**です。

### 序盤の基本方針

- 弱いハンドで大きなポットを作らない
- ポジションの良い場面で参加する
- 相手のクセを観察する
- 強いハンドではしっかりバリューを取る
- ブラフは控えめにする

### 観察ポイント

- よく参加する人は誰か
- すぐ降りる人は誰か
- 大きく打つ人は本当に強いのか
- ショートスタックになった時に焦りそうな人は誰か
- バブルで固くなりそうな人は誰か

序盤の観察は、終盤の武器になります。

### 序盤でチップを増やす技術

序盤でチップを増やす目的は、無理に大きな勝負をすることではありません。

大切なのは、同卓しているプレイヤーを観察し、「誰から、どんな形でチップを取れるか」を見つけることです。

序盤は、テーブル全体をスキャンする時間です。

カードだけを見るのではなく、人を見ます。

### 観察から戦術を作る流れ

1. 相手の参加頻度を見る
2. ショーダウンしたハンドを覚える
3. ベットサイズのクセを見る
4. フォールドできる人か、コールしすぎる人かを分ける
5. その人に合った取り方を決める

この流れができると、序盤のハンド選択がただの暗記ではなく、テーブル攻略になります。

### ラベリング：誰からチップを取るかを決める

序盤の観察では、相手を細かく完璧に分析する必要はありません。

まずはシンプルに、同卓者を次の2つに分けます。

- チップを取りやすい人
- チップを取りにくい人

このラベリングができると、無駄な勝負が減り、勝てる相手に集中できます。

ポーカーは、全員を倒すゲームではありません。

**自分が勝ちやすい相手から、必要なチップを集めるゲームです。**

### チップを取りやすい人の特徴

| 見極めポイント | 具体的なサイン | 狙い方 |
|---|---|---|
| コールが多い | フロップ、ターンで簡単に降りない | 強い手で大きめに打つ |
| ハンドを見せがち | ショーダウンまで行く回数が多い | レンジを読みやすい |
| ベット額が正直 | 強い時は大きく、弱い時は小さい | サイズで強弱を判断する |
| ポジションを気にしない | 不利な位置から広く参加する | 良いポジションで孤立させる |
| 感情が出る | 負けた後に参加が増える | 強い手で待ち受ける |
| ドローを追いすぎる | オッズに合わないコールをする | 完成前に高く払わせる |

このタイプには、難しいブラフよりもバリューが効きます。

相手が降りないなら、降ろそうとするのではなく、払ってもらいます。

### チップを取りにくい人の特徴

| 見極めポイント | 具体的なサイン | 対応 |
|---|---|---|
| 参加ハンドが固い | ほとんど入らない | 大きな抵抗は信用する |
| ポジションを理解している | 後ろの位置から攻めてくる | 無理に戦わない |
| ベットサイズが安定している | 強弱がサイズに出にくい | 情報が少ないので慎重に |
| フォールドできる | セカンドペアや弱いドローを捨てる | バリューが取りにくい |
| 3ベットが的確 | こちらのオープンに圧をかける | 広げすぎず、強い手で受ける |
| スタックを守れる | 不要なオールインをしない | 序盤に大きくぶつからない |

このタイプから無理に取ろうとすると、自分の方が難しい判断をさせられます。

序盤は、強い相手と正面衝突するより、取りやすい相手を探す方が利益になります。

### 見極める時のチェックリスト

毎周、次の質問を自分に投げます。

- この人はどれくらい参加しているか
- コールが多いか、フォールドが多いか
- ショーダウンで何を見せたか
- 強い時と弱い時でベットサイズが変わるか
- ポジションを意識しているか
- 負けた後にプレイが荒くなるか
- 自分のベットに素直に降りるか
- 大きなポットで冷静に判断できるか

このチェックを続けると、テーブルがただの9人席ではなく、攻略マップに変わります。

### ラベル別の基本方針

| ラベル | 方針 |
|---|---|
| 取りやすい人 | バリュー中心でポットを作る |
| 降りすぎる人 | 小さく何度も拾う |
| 攻撃的な人 | 強い手で受ける |
| 読みにくい人 | 大きな勝負を避ける |
| 強い人 | 序盤は無理に戦わない |

大切なのは、相手全員に同じ戦い方をしないことです。

**相手をラベリングし、取りやすい人から取る。**

これが序盤のチップ構築の基本になります。

### プレイヤータイプ別の序盤戦術

| 相手タイプ | 特徴 | 取り方 | 注意点 |
|---|---|---|---|
| コールしすぎる人 | ワンペアやドローでついてくる | 強いハンドで大きめにバリューを取る | ブラフを減らす |
| すぐ降りる人 | フロップやターンで諦めやすい | ポジションから小さくポットを拾う | 強く抵抗されたら無理しない |
| 大きく打つ人 | ベットサイズが大きくなりやすい | 強い手で受ける、罠を張る | 薄いコールをしない |
| 参加しすぎる人 | プリフロップで広く入る | 良いポジションで孤立させる | 多人数ポットにしない |
| 固すぎる人 | 強い手しか参加しない | ブラインドや小さいポットを狙う | 大きなポットでは信用する |
| 怖がりな人 | 失うことを嫌がる | 継続ベットやターン圧力を使う | コールされた時は慎重にする |

### 序盤の稼ぎ方は3種類

序盤で増やす方法は、大きく3つです。

| 稼ぎ方 | 狙う相手 | やること |
|---|---|---|
| バリューで取る | コールしすぎる人 | 強い手でしっかり打つ |
| 小さく拾う | すぐ降りる人 | ポジションから小さくスチールする |
| 罠で取る | 攻撃的な人 | 強い手で相手に打たせる |

この3つを混ぜると、序盤のチップ増加が安定します。

ただし、日本型の低ITM率トーナメントでは、序盤で平均以上のスタックを作る価値が高くなります。

だからこそ、序盤は「無理をしない」だけでは足りません。

**観察して、相手ごとに取り方を変える。**

これが、インマネ率を上げるための最初の武器になります。

## 2. 中盤：スタックサイズ別に戦い方を変える

中盤からはブラインドとアンティが重くなり、何もしないだけでチップが減っていきます。

ここからは「良いカードを待つゲーム」ではなく、**自分のスタック量に合った作戦を選ぶゲーム**になります。

### 40BB以上：選択肢が多い状態

余裕があります。無理なオールイン勝負を避けながら、ポジションを使って小さくチップを集めます。

やること:

- ボタン、カットオフから積極的にスチールする
- 強い相手と大きなポットを作りすぎない
- ショートスタックへのコールは慎重にする
- 自分より大きいスタックとの衝突を避ける

### 20BBから40BB：インマネ狙いの主戦場

最も判断力が問われるゾーンです。

まだ余裕はありますが、無駄な参加を続けると一気に危険水域へ落ちます。

やること:

- オープンするハンドを少し絞る
- 3ベットオールインされそうな相手を意識する
- スチールできる相手を狙う
- コールよりもレイズで主導権を取る

### 10BBから20BB：押すか降りるかを明確にする

このゾーンでは、だらだらコールするのが最も危険です。

中途半端に参加してフォールドすると、次の勝負の燃料がなくなります。

やること:

- オープンレイズ後に降りる前提の参加を減らす
- オールインできるハンドを準備する
- 後ろにいる相手のスタックを確認する
- フォールドエクイティが残っているうちに動く

### 10BB未満：待ちすぎない

ここでは、完璧なハンドを待ちすぎるとブラインドで消えてしまいます。

大切なのは「勝負できる手で先に入ること」です。

やること:

- 先にオールインして相手に判断させる
- コールオールインより、プッシュオールインを優先する
- ボタン、スモールブラインド、カットオフのチャンスを逃さない

---

## 3. バブル直前：最大の罠と最大のチャンス

インマネ直前を「バブル」と呼びます。

ここはトーナメントで最も空気が変わる場所です。

多くの人が「ここまで来たら飛びたくない」と考えます。その心理が、最大の罠にも、最大の武器にもなります。

### バブルでやってはいけないこと

- 自分より大きいスタックに軽く突っ込む
- ショートスタックなのに無限に待つ
- ミドルスタックで無謀なコールをする
- 「インマネしたい」だけでブラインドを払い続ける

### バブルで狙うべきこと

- 自分より小さいスタックに圧をかける
- インマネを強く意識している相手のブラインドを狙う
- ビッグスタックとは大きな衝突を避ける
- ショートスタックが複数いる時は無理に勝負しない

バブルは、全員が同じカードで戦っているように見えて、実は「恐怖の量」が違います。

その恐怖を読める人が、チップを拾えます。

---

## 4. スタック別バブル戦略

### ビッグスタック

あなたはテーブルの重力です。

相手は飛びたくないので、普段より多く降ります。

やること:

- ミドルスタックにプレッシャーをかける
- ブラインドスチールを増やす
- ショートスタックへの無謀なコールは避ける
- 自分より大きいスタックとは慎重に戦う

### ミドルスタック

最も注意が必要です。

インマネ目前でミドルスタックが飛ぶのは大きなミスになりやすいです。

やること:

- ビッグスタックとの大勝負を避ける
- ショートスタックが複数いるなら我慢する
- 自分より小さいスタックには圧をかける
- コールで大きく賭けるより、先に攻める

### ショートスタック

ただ耐えるだけでは、ブラインドに削られて終わります。

まだ相手が降りてくれるうちに勝負します。

やること:

- ボタン、カットオフ、スモールブラインドから押す
- 自分のビッグブラインドが来る前にチャンスを探す
- 先にオールインしてフォールドエクイティを使う
- コールで全チップを入れる場面は慎重に選ぶ

---

## 5. インマネ率を上げる実戦チェックリスト

### 毎ハンド確認すること

- 自分のスタックは何BBか
- 後ろにショートスタックはいるか
- 後ろにビッグスタックはいるか
- 自分が飛ぶとインマネにどれくらい影響するか
- 相手はインマネを怖がっているか
- この勝負は本当に必要か

### 参加前の合言葉

**このハンドは、今の自分の作戦に合っているか？**

強そうなカードでも、状況に合っていなければ危険です。

逆に、そこそこのカードでも、相手と状況が良ければチャンスになります。

---

## 6. よくある失敗

### 失敗1：序盤に謎の大勝負をする

序盤で飛ぶと、トーナメントの後半にあるおいしい局面に参加できません。

強いハンドでも、相手の強さとポットサイズを見ながら進めます。

### 失敗2：中盤でコールが増える

コールは一見安全に見えますが、主導権がありません。

中盤以降は、参加するならレイズ、勝負しないならフォールドを明確にします。

### 失敗3：バブルで固まりすぎる

インマネしたい気持ちが強すぎると、全員から狙われます。

守るだけではなく、相手が守りに入った瞬間を狙います。

### 失敗4：ショートスタックを待ちすぎる

「あと少し待てばいい」と思っているうちに、フォールドエクイティが消えます。

まだ相手を降ろせるうちに動くことが大切です。

## 7. 練習メニュー

### レベル1：BB換算を習慣にする

チップ枚数ではなく、常に「自分は何BB持っているか」で考えます。

例:

- 60,000点持ち
- ビッグブラインドが3,000点
- 自分は20BB

この感覚が身につくと、判断が一気に速くなります。

### レベル2：相手を3タイプに分ける

テーブルの相手を次の3つに分類します。

- 固い人
- よく参加する人
- プレッシャーに弱そうな人

バブルでは、特に「プレッシャーに弱そうな人」が宝箱になります。

### レベル3：バブル想定で練習する

練習では、毎回この状況を想定します。

- 残り人数：20人
- インマネ：18人
- 自分：18BB
- テーブルに5BBのショートが2人いる

この時、自分は無理に大勝負すべきではありません。

しかし、全員が固まっているなら、小さなスチールは有効です。

## 8. インマネ直前の判断早見表

| 自分の状態 | 基本方針 | 避けたい行動 |
|---|---|---|
| ビッグスタック | 小さく圧をかける | 無謀なコールで倍付けする |
| ミドルスタック | 飛ばない勝負を選ぶ | ビッグスタックと正面衝突する |
| ショートスタック | 先にオールインする | ブラインドで自然消滅する |
| 周りにショートが多い | 我慢も武器にする | 不要なオールイン勝負をする |
| 周りが全員固い | スチールを増やす | 強い手だけを待ち続ける |

## 9. 日本型ITMとレジクロ参加の考え方

海外の大型トーナメントでは、ITMが全体の12%から15%前後に設定されることが多くあります。

一方で、日本のアミューズ系・権利型トーナメントでは、実質的な入賞や権利獲得が上位3%から8%前後になることがあります。

この差は、戦略を大きく変えます。

### 海外型ITMと日本型ITMの違い

| 形式 | ITMの目安 | 基本戦略 |
|---|---:|---|
| 海外大型MTT | 12%から15%前後 | バブルを越える設計が重要 |
| 日本型トーナメント | 3%から8%前後 | バブルに着く前に戦えるスタックを作る |

海外型では、ある程度ショートでもバブルまで届くことがあります。

しかし日本型では、ただ耐えるだけでは上位数%に届きにくくなります。

つまり、日本型の合言葉はこうです。

**インマネ前に、武器を作れ。**

### ITM率で変わる戦略

ITM率が違うと、同じ「インマネ狙い」でもゲームの目的が変わります。

海外型の12%から15%ITMは、バブルまでの距離が比較的近いため、無理な衝突を避けながら到達する戦略が成立しやすくなります。

日本型の3%から8%ITMは、バブルまでの距離が遠いため、ただ生き残るだけでは足りません。中盤までに、相手へ圧をかけられるスタックを作る必要があります。

| 局面 | 海外型12%から15%ITM | 日本型3%から8%ITM |
|---|---|---|
| 序盤 | 事故回避と観察を優先する | 事故回避しながら、弱い相手からチップを取る |
| 中盤 | スタック維持とスチールを重視する | 平均以上のスタックを作るために勝負所を増やす |
| バブル前 | 飛ばない判断を重視する | ショートで到達しないように早めに仕掛ける |
| バブル | 固い相手に圧をかける | 圧をかける側のスタックで到達する |
| ショート時 | 耐える選択肢も残る | 待ちすぎると届かないため、先に動く |

### ITM率別の目標スタック

ITM率が高い大会では、平均以下でもバブルに近づける場面があります。

しかしITM率が低い大会では、平均以下で終盤に入ると、ほとんどの場合は「降りて祈る」か「不利なオールインを受ける」展開になります。

| ITM率 | 目標イメージ | 重要なこと |
|---:|---|---|
| 15%前後 | バブルまで生き残る | 大事故を避け、ミドルスタックを保つ |
| 10%前後 | 守りと攻めのバランス | 中盤でスチールを増やす |
| 5%前後 | 上位争いに残る | 序盤から中盤で平均以上を目指す |
| 3%前後 | ファイナルテーブル候補になる | 早い段階でダブルアップや大きなポット獲得が必要 |

### 日本型で変えるべき打ち方

日本型の低ITM率では、次のように考え方を変えます。

- 「飛ばない」より「後で圧をかけられるスタックを作る」
- 序盤のルースな相手からバリューを取り切る
- 中盤でスチールとリスチールを増やす
- ショートになる前に勝負する
- バブルに入ってから頑張るのではなく、バブル前に戦える状態を作る

日本型では、バブルはゴール前ではありません。

**バブルは、準備できた人だけが武器を使える最終ステージです。**

### レジクロ参加は本当に有利か

「レジクロで入る方がICMが高い」と言われることがあります。

これは半分本当です。

レジクロで入ると、すでに飛んだプレイヤーがいるため、同じ参加費でも入賞までの距離は短くなります。序盤の事故も避けられます。

そのため、ICMだけで見ると、スタートチップの価値が高く見える場面があります。

ただし、インマネ率を上げるという目的では注意が必要です。

### レジクロ参加のメリット

- 序盤の事故を避けられる
- 残り人数が減った状態から始められる
- ショートスタック戦が得意なら勝負しやすい
- 海外型の15%ITMならミニマムキャッシュ狙いが成立しやすい

### レジクロ参加の弱点

- 開始時点でスタックが浅くなりやすい
- ポストフロップで実力差を出す時間が少ない
- 1回のオールイン勝負への依存度が高くなる
- ビッグスタックに圧をかけられる側になりやすい
- 日本型の上位3%から8%ITMでは、ショートで入っても届きにくい

### 早め参加が有利になる条件

早め参加が有利になるのは、序盤から相手のミスを拾える時です。

特に次のような環境では、最初から入る価値が高くなります。

- コールしすぎるプレイヤーが多い
- 強いハンドにしっかり支払ってくれる相手がいる
- ポストフロップで差を出せる
- 序盤のテーブル観察を後半に活かせる
- 上位5%前後を狙う必要がある

レジクロ参加は近道に見えます。

しかし日本型ITMでは、近道ではなく「短距離の崖登り」になることがあります。

### 判断早見表

| 目的 | 有利になりやすい入り方 |
|---|---|
| 序盤事故を避けたい | レジクロ参加 |
| ミニマムITMだけ狙う | レジクロ参加もあり |
| 上位3%から8%を狙う | 早め参加 |
| 実力差でチップを増やしたい | 早め参加 |
| ショートオールイン戦が得意 | レジクロ参加もあり |
| ポストフロップで差を出したい | 早め参加 |

結論はシンプルです。

**レジクロ参加はICM効率を上げることがあるが、スタック構築の時間を失う。**

**日本型ITMでは、早めに参加して中盤までに平均以上のスタックを作る方が、インマネ率を上げやすい場面が多い。**

## 10. 最後に

インマネは、ただの結果ではありません。

それは「焦りをコントロールし、相手の心理を読み、必要な時に一歩前へ出る」ことで近づく到達点です。

トーナメントのテーブルは、小さな宇宙船のようなものです。

燃料はチップ。

地図はスタックサイズ。

追い風は相手の恐怖。

そして船長は、あなたです。

インマネを目指すなら、毎ハンドこう問いかけてください。

**これは生き残るための一手か。未来のチップを増やす一手か。**

その問いを持てるだけで、あなたのトーナメントは一段強くなります。
